# Sundown Studio Clone

A responsive front-end clone inspired by the **Sundown Studio** website. This project is focused on practicing **modern HTML/CSS layout**, clean UI structure, and light **Vanilla JavaScript** interactions.

> **Status:** Frontend-only (no backend)

## Features

- Responsive layout (mobile / tablet / desktop)
- Modern UI styling with CSS
- Reusable page sections and components
- Lightweight JavaScript for interactivity (where needed)
- Clean HTML structure and semantic markup

## Built With

- **HTML** (≈42%)
- **CSS** (≈51.5%)
- **JavaScript** (≈6.5%)

## Getting Started

### 1) Clone the repository
```bash
git clone https://github.com/shahmeerking231/sundown_studio_clone.git
cd sundown_studio_clone
```

### 2) Run locally

#### Option A: Open directly
Open `index.html` in your browser.

#### Option B: Run with a local server (recommended)

**VS Code Live Server**
- Install the **Live Server** extension
- Right-click `index.html` → **Open with Live Server**

**Python**
```bash
python -m http.server 5500
```
Then visit:
- `http://localhost:5500`
